# nougyoushiyou 農業する人が増えてほしい 衰退していく一方の日本農業 　

衰退していくの日本農業

農業従事者の推移と高齢化

近年、日本全国では農業従事者の減少が大きな問題となっている。 １９８５年頃をピークに減少が著しくなり、年を重ねるごとに高齢化もすすんできた。

これからのカギとなるスマート農業

農業従事者の減少と高齢化がすすんだことにより、一人当たりの耕地面積は拡大しまっている。 この現状がすすみつづけると日本農業は破綻してしまうだろう。 そこで、近年ではスマート農業がわずかだが始まっている。 スマート農業が発展すると農業従事者が少なくてもより広大な耕地面積を面倒見ることが可能になる。

スマート農業の例

  • 自動収穫機械（トマトやキャベツ）
  • ドローンを使った散布作業
  • 農業用アシストスーツ
